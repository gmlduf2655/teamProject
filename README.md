<h6>기획의도</h6>
<p>
	실제 운영중인 영화 예매 사이트의 구조를 벤치마킹하여 현업에서 사용하는 프로세스를 경험해보고 비즈니스를 이해하는데 목적을 두고 개발하였습니다.
